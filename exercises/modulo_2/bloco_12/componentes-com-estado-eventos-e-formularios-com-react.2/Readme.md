## Bloco 12 - Componentes com Estado, Eventos e Formulários com React
### Dia 2 - Formulários no React

O objetivo dos exercícios desse dia foi colocar em prática o que havia aprendido em React sobre a criação de formulários refazendo o exercício feito anteriormente usando JavaScript "clássico", disponível [nesse link](https://github.com/gabrielanebbia/trybe-exercises/pull/20/files)

Os requisitos do exercí­cio foram:

**Exercício 1**
Cria um <fieldset /> para os dados pessoais a seguir
- [ ] Nome - Texto
- [ ] Email - Texto
- [ ] CPF - Texto
- [ ] Cidade - Texto
- [ ] Estado - ComboBox
- [ ] Tipo - Radio Button

**Exercício 2**
Cria outro <fieldset /> para dados do seu último emprego:
- [ ] Resumo do currículo - TextArea
- [ ] Cargo - TextArea
- [ ] Descrição do cargo - Texto

**Exercício 3**
- [ ] Cria um botão que monta uma <div> com o consolidado dos dados que foram inseridos no formulário.

**Exercício 4**
- [ ] Cria um botão Limpar que limpa todos os campos do formulário e a <div> com seu currículo também.
